# whitelist-battery

su環境用のアプリです
ユーザー or システムの
選択したpackageのトグルをon off で切り替えることにより
バッテリー最適化を無効に追加したり出来ます。

## 概要

- アプリ一覧ボタンを押して表示されたpackageから
- 選択したpackageのトグルをon off で切り替えることにより
- バッテリー最適化無効に追加したり出来ます。

> 基本的な端末では利便性が皆無のアプリです()
> 本体設定から変更できない or shellから面倒なsu ユーザー向けです

> バイナリのsource https://github.com/coara-chocomaru/setappops
